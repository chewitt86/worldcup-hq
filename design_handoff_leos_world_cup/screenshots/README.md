# Screenshots — visual targets

These are reference captures of the **mobile (~390px)** layout. Match these pixel-for-pixel,
then verify the tablet/desktop (≥760px) multi-column layouts described in the README.

| File | Screen |
|---|---|
| `01-home.png` | Home — jumbotron countdown, LED ticker, live leaders, next-up |
| `02-sweepstake.png` | Sweepstake — podium + ranked person cards (note the "OUT OF THE SWEEPSTAKE" stamp on eliminated players) |
| `03-teams.png` | Teams — tier filter + nation grid (note the corner "OUT" ribbon on knocked-out teams) |
| `04-groups.png` | Groups — live tables A–H with owner avatars + leaders |
| `05-knockout.png` | Knockout — champion hero, jump-to, two-sided bracket on the pan/zoom canvas |
| `06-map.png` | Map — default view, all arcs to host venues |
| `07-map-team-selected.png` | Map — single team selected (clean view: only that team's arcs/dots) + games card below |
| `08-admin-login.png` | Admin — PIN gate ("GROWN-UPS ONLY", demo PIN 1966) |
| `09-admin.png` | Admin — dashboard (players, settings, API keys, results, edit team data) |
| `10-team-popup.png` | Team detail popup (group form, results, backers, FIFA squad link) |
| `11-game-modal.png` | Knockout game modal (owners + kickoff BST/venue or score) |

Tip: the bracket and map are zoomable canvases — these captures show the default fit. Scroll/zoom
in the live prototype (`Leo's World Cup.html`) to inspect detail.
